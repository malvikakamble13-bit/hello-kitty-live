# 🎀 Hello Kitty Live

A cute, interactive Hello Kitty page — her eyes follow your cursor wherever it moves on the screen.

## 📖 About

A small, playful front-end project focused on real-time interactivity. As you move your mouse around the page, Hello Kitty's eyes track the cursor position, giving the illusion that she's watching you move.

## ✨ Features

- Real-time cursor tracking
- Smooth eye movement animation
- Simple, clean, pure front-end implementation (no frameworks)

## 🛠️ Built With

- HTML
